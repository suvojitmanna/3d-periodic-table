💎 Overview

A modern and interactive 3D periodic table built using pure HTML, CSS, and JavaScript.
This project transforms the traditional periodic table into a visually immersive and dynamic web experience with smooth animations and elegant UI.

✨ Features
⚛️ Interactive Elements
Explore all chemical elements visually
🎨 Modern UI Design
Clean layout with glow effects and smooth styling
⚡ Smooth Animations
Dynamic transitions between layouts
🔄 Multiple Layout Modes
Table
Sphere
Helix
Grid
🖱️ User Interaction
Drag to rotate
Scroll to zoom
📱 Responsive Design
Works across all devices
